## Filter Workflow

```
User Flow:
1. Click "Filters" button
   â†“
2. Filter panel slides down (inline, not overlay)
   â†“
3. Click any filter dropdown
   â†“
4. Search (if applicable) and check/uncheck options
   â†“
5. Click "Select" to close dropdown (or "Cancel" to discard)
   â†“
6. Repeat for other filters
   â†“
7. Click "Filter" button to apply ALL selections
   â†“
8. Table updates with filtered data
   â†“
9. Click "Close" to hide filter panel (filters remain active)
```

## Color Scheme

```

- Headers: Bold, dark
- Labels: Semi-bold, purple (#2a2a85)
- Links: Blue (Reset), Red (Clear All)
```
